React Web Component Proof-of-Concept

This demonstrates how to add an existing React component to a microfrontend without causing style conflicts.

There are two buttons here. The first one is a pure HTML button inside the parent app which has red color and squared corners. The second one is a React button component rendered inside custom element which has blue color and rounded corners.

We have two separate stylesheets, one for the parent app and one that is embedded within the React component. Both buttons are styled using the button selector in CSS.

Since we are rendering the React button component inside custom element, you might be wondering how to change the props. Props of the component can be changed from parent app by using the following code: 

document.querySelector('react-button').setAttribute('color', 'green');
